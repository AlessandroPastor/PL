Ejemplos predefinidos
Problema de Producción
Una fábrica produce dos productos con diferentes tiempos de producción y ganancias.
Maximizar: 40x + 30y
Sujeto a:
2x + y <= 1000 (Tiempo máquina A)
x + 2y <= 800 (Tiempo máquina B)
x >= 0, y >= 0
Problema de Dieta
Minimizar el costo de una dieta que cumpla con requerimientos nutricionales mínimos.
Minimizar: 0.6x + y
Sujeto a:
10x + 4y >= 20 (Proteínas)
5x + 5y >= 20 (Carbohidratos)
2x + 6y >= 12 (Vitaminas)
x >= 0, y >= 0
Problema de Transporte
Minimizar los costos de transporte entre fábricas y almacenes.
Minimizar: 5x + 4y + 3z + 6w
Sujeto a:
x + y = 100 (Demanda Almacén 1)
z + w = 150 (Demanda Almacén 2)
x + z <= 120 (Suministro Fábrica A)
y + w <= 180 (Suministro Fábrica B)
x >= 0, y >= 0, z >= 0, w >= 0
Información importante

Este optimizador resuelve problemas de programación lineal usando:

  • Método gráfico para problemas con 2 variables
  • Método Simplex para problemas con más variables
  • Análisis de sensibilidad para evaluar cómo cambia la solución con variaciones en los parámetros

Para problemas con más de 4 variables, solo estará disponible el método Simplex.

Ayuda del Optimizador

¿Cómo usar esta herramienta?

Esta herramienta te permite resolver problemas de programación lineal de forma gráfica (para 2-3 variables) o mediante el método Simplex (para más variables).

Formato de entrada:
  • Función objetivo: Ingresa una expresión lineal como "3x + 5y - 2z"
  • Restricciones: Usa los operadores <=, >= o =. Ejemplo: "2x + 3y <= 10"
  • Variables: Usa letras como x, y, z, w. No uses números en los nombres (como x1, x2)
Métodos de solución:
  • Solución Gráfica: Visualización interactiva para problemas con 2-3 variables
  • Método Simplex: Algoritmo para problemas con cualquier número de variables
  • Análisis de Sensibilidad: Evalúa cómo cambios en los coeficientes afectan la solución
Ejemplos válidos:
Maximizar: 3x + 5y
Sujeto a:
x + y <= 10
2x - y >= 4
x >= 0, y >= 0
Minimizar: 4x + 3y + 2z
Sujeto a:
x + 2y + z >= 5
3x + y + 2z <= 12
x >= 0, y >= 0, z >= 0
Limitaciones:
  • La visualización gráfica solo está disponible para problemas con 2-3 variables
  • Para problemas con más de 4 variables, solo está disponible el método Simplex
  • No se admiten restricciones no lineales o variables enteras
